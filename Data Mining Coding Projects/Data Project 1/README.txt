HOW TO RUN:
-----------
Usage:
    python apriori.py <input_file> <min_support> <output_file>

Parameters:
    input_file   - Path to the transaction dataset file
    min_support  - Minimum support count (positive integer)
    output_file  - Path to the output file for results

Example:
    python apriori.py T10I4D100K.txt 500 result500.txt


SYSTEM REQUIREMENTS:
--------------------
- Python 3.7 or higher
- NumPy library (install via: pip install numpy)
